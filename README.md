# python-BasicLearning

Python 基础语法学习示例集合，包含基础语法、函数、数据结构、面向对象、异常、模块、文件操作和高阶语法练习。

## 环境要求

- Python 3.8+ 推荐
- 少量旧示例可能来自 Python 2 语法，如果运行失败，请按文件内容调整解释器或语法

## 安装依赖

本项目只使用 Python 标准库，`requirements.txt` 中不需要安装第三方包。

## 运行示例

```bash
python3 "01_python基础/py_01_helloworld_1.py"
python3 "01_python基础/py_16_名片管理系统.py"
python3 "02_面向对象/py_01_第一个面向对象.py"
```

## 目录说明

- `00_笔记`：学习笔记和图解
- `01_python基础`：变量、运算符、输入输出、条件、循环、函数和基础数据结构
- `02_面向对象`：类、对象、封装、继承、多态、异常、模块和文件
- `03_高阶语法`：标准输入等进阶示例

## 开发建议

本项目是脚本集合，不需要提交虚拟环境。若需要安装额外依赖，建议在本地创建 `.venv` 并自行维护 `requirements.txt`。
